# Bookends
A full-stack book collection manager and cataloger inspired by Goodreads. Built with a modern tech stack to handle high-performance searching, custom user collections, and provides personalized recommendations.

## Tech Stack
- Frontend: Next.js 16+ (App Router), TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL (Relational data & persistence)
- Caching: Redis
- Authentication: Passport.js and Google OAuth 2.0

## Features
- Instant setup with Google OAuth 2.0 integration
- Search millions of titles using the Google Books API
- Create customed themed shelves like "Best of 2026" or "Classics"
- Receive personalized book recommendations based on titles you've read
